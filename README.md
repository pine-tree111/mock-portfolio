# 🤖 Accounting AI Systems

AI-powered automation tools for UK accounting firms. Built with Python, Claude API, n8n, and LangGraph.

## 🎯 What This Is

A portfolio of AI agents and automation tools designed to streamline accounting workflows:

- **Invoice Processing** — Extract and categorise invoice data automatically
- **Transaction Analysis** — Categorise and summarise bank transactions
- **Document Intelligence** — Parse PDFs, receipts, and bank statements
- **Automated Reporting** — Generate client-ready financial summaries

## 🛠 Tech Stack

- **Python 3.12+** — Core language
- **Claude API** — AI-powered document understanding
- **n8n** — Workflow automation
- **LangGraph** — Agent orchestration
